Oracle Redwood Design System
How this portfolio is built on Oracle's design philosophy

Oracle Redwood is Oracle's human-centered design system — built around clarity, accessibility, and consistency across enterprise applications. This portfolio deliberately adopts Redwood's visual language and UX patterns to demonstrate real-world Oracle Cloud UI/UX fluency.

Design Principles Applied
🎯 Clarity First
Every section has a clear information hierarchy — section labels, titles, descriptions — with no visual clutter competing for attention.
⚖️ Consistent Scale
Spacing, border-radius, and type sizes follow a strict scale — 8px base grid — matching Redwood's spatial rhythm.
🌈 Purposeful Color
Redwood Red (#C74634) used exclusively for primary actions and emphasis — never decoratively. Neutrals carry everything else.
🤝 Human-Centered
Progressive disclosure throughout — teasers, expand/collapse, and "view all" patterns reduce information overwhelm.

Redwood Components & Patterns
Tab Bar Navigation
Oracle JET oj-tab-bar with Redwood dark header — matching Oracle Fusion ERP navigation. Active tab underlined in Redwood Red.
Card Surface
White cards on grey page background — the core Redwood surface pattern. 14px border-radius and 1px border, never hard shadows.
Section Labels
Uppercase tight-tracked labels above every section — mirroring Oracle Fusion's panel header pattern used in VBCS form layouts.
Left Border Accent
Featured cards use a 4px Redwood Red left border — the standard Redwood highlight for selected or featured items.
Skill Progress Bars
Styled after Oracle JET oj-progress-bar — neutral grey track, Redwood Red fill.
Pill / Tag Chips
Tech and interest tags styled as Oracle JET oj-chip — dot indicator, hover matching Redwood chip interactions.
SPA Routing
Built on Oracle JET's oj-module + CoreRouter — same architecture as Oracle VBCS applications.
